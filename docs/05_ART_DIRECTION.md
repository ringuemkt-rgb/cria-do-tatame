# 05 - Direção de Arte

## Estilo

HD Pixel Art 2.5D com pintura digital controlada, leitura mobile e atmosfera premium.

## Identidade

A identidade precisa ser reconhecível mesmo sem ler o nome:

- Gorila Silverback.
- Coroa.
- Faixa preta.
- Tatame azul/dourado.
- Preto fosco.
- Dourado queimado.
- Branco sujo.
- Vermelho conflito.
- Azul rio profundo.
- Verde mangue.

## Paleta oficial

| Cor | Hex | Uso |
|---|---:|---|
| Preto absoluto | `#0A0A0A` | fundos, menus, sombra |
| Preto fosco | `#1A1A1A` | painéis, HUD, cards |
| Dourado queimado | `#B8860B` | honra, bordas, faixa |
| Amarelo honra | `#F2C230` | CTA, seleção, vitória |
| Branco sujo | `#F2F2F2` | texto principal, gi, logo |
| Vermelho conflito | `#D92323` | dano, crise, perigo |
| Azul rio | `#1E3A5F` | água, noite, circuito oficial |
| Verde mangue | `#2D5016` | raiz, natureza, recuperação |
| Roxo sombra | `#4B0082` | dualidade e tensão moral |

## UI

- Mobile-first.
- Botões grandes.
- Safe area.
- Alto contraste.
- Bordas douradas.
- Fundo escuro.
- Tipografia condensada e forte.

## Arenas de referência visual

As imagens de referência atuais definem duas linhas:

1. **Arena do Dique / Salvador:** ginásio oficial, telões, tatame azul/dourado, público, escala de campeonato.
2. **Terreiro da Luta / Ituberá:** madeira, mangue, sol baixo, rio, ancestralidade, treino raiz.

## Proibição visual

- Não usar gorila infantil.
- Não usar cartoon leve.
- Não usar excesso de cores.
- Não usar efeito plástico barato.
- Não copiar assets comerciais.
