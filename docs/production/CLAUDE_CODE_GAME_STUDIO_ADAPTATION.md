# Adaptacao Claude Code Game Studios para Cria do Tatame

## Analise da referencia

O repositorio Claude Code Game Studios organiza uma sessao de IA como um estudio de desenvolvimento com agentes, skills, hooks, regras e templates. A proposta original declara 49 agentes, 73 skills, 12 hooks, 11 regras e 41 templates para cobrir design, programacao, arte, audio, narrativa, QA e producao.

## O que aproveitamos

- Hierarquia de diretores, lideres e especialistas.
- Separacao clara de responsabilidades.
- Quality gates por area.
- Workflows por fase do projeto.
- Regras por pasta.
- Templates para GDD, arquitetura, sprint, HUD e QA.
- Sistema de revisao antes de finalizar entregas.

## O que adaptamos para nosso jogo

O Cria do Tatame precisa de dominios proprios:

- jiu-jitsu brasileiro;
- biomecanica e cinesiologia;
- Godot 4.2+;
- sprites 2D pixel art;
- analise de videos autorizados;
- Baixo Sul da Bahia;
- modo carreira em portugues brasileiro;
- IA de rivais por gameplan;
- combate posicional.
