# PT-BR Runtime Lock v0.1

## Diretriz definitiva

Todo texto visivel ao jogador deve estar em portugues brasileiro.

## Aplicado nesta etapa

- Catalogo de tecnicas em PT-BR.
- Dialogos de Mestre Dende, Davi, Cassio e Tinker Bell.
- Menu principal.
- Hub Terreiro da Luta.
- Cena de combate.
- Labels de estado.
- Cria Live.
- Manifesto de audio.
