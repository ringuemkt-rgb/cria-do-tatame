# Fonte visual - mapa do Baixo Sul v01

Mapa artístico original produzido pela ferramenta de imagem integrada da OpenAI. A arte
não pretende substituir cartografia oficial: serve como placa visual de meta-mapa para
hubs densos. Coordenadas, nomes, custos, desbloqueios e rotas permanecem nos JSONs e são
desenhados pelo Godot.

Antes de release, a distribuição visual deve ser confrontada com a malha municipal do
IBGE/QGIS, como já exige `data/world_map_geo.json`.
