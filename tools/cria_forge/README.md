# Cria Forge CLI

Ferramenta local simples para validar dados e gerar pacotes de tecnica para o Cria do Tatame.

## Comandos

```bash
python tools/cria_forge/cria_forge.py validate
python tools/cria_forge/cria_forge.py technique baiana
```

## Saidas

- relatorio de validacao;
- pasta de tecnica;
- sprite_request.json;
- qa_report.md.

## Papel

Esta ferramenta nao substitui Godot. Ela prepara dados e relatorios para o pipeline do jogo.
