# Technique Pack Template

Use esta pasta como referencia para cada tecnica nova.

## Estrutura

```txt
<technique_id>/
  briefing.md
  biomecanica.md
  gameplay_card.json
  sprite_request.json
  manifest_stub.json

```

## Fluxo

1. preencher briefing;
2. validar BJJ;
3. gerar gameplay_card;
4. gerar sprite_request;
5. criar manifest;
6. rodar QA.
